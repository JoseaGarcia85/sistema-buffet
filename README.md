# Sistema Buffet

Proyecto en Java para la gestión de un buffet escolar.

## Descripción
Este software permite gestionar ventas, compras, inventario y reportes de un buffet. Incluye interfaz gráfica con calendario y selectores de fecha, así como conexión a base de datos MySQL.

## Requisitos
- Java 17
- MySQL (con la base de datos `buffet` creada)
- Librerías:
  - JCalendar
  - FlatLaf
  - LGoodDatePicker
  - DateTimePicker
  - MySQL Connector/J

## Instalación y ejecución

### Desde NetBeans
1. Clonar el repositorio:
   ```bash
   git clone https://github.com/JoseaGarcia85/sistema-buffet.git
Abrir el proyecto en NetBeans.

Asegurarse de tener configuradas todas las librerías mencionadas.

Compilar y ejecutar el proyecto.
Usando el JAR compilado

Clonar el repositorio o descargar el JAR:

git clone https://github.com/JoseaGarcia85/sistema-buffet.git


Abrir terminal en la carpeta target.

Ejecutar el JAR:

java -jar sistema-buffet-1.0.jar


Configurar la base de datos buffet en MySQL antes de ejecutar.

Estructura del proyecto

src/: Código fuente en Java.

target/: Archivos compilados (JAR generado por Maven).

pom.xml: Archivo de configuración de Maven.

README.md: Información del proyecto.
